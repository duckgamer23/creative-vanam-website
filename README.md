# CreativeVanam - Digital Agency Website

A modern, responsive website for CreativeVanam digital agency. Built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features

- **Modern Design**: Dark theme with neon green accents
- **Fully Responsive**: Mobile-first approach with optimized layouts
- **Interactive Elements**: Smooth animations, hover effects, and mobile menu
- **Fast Performance**: Optimized assets and minimal dependencies
- **SEO-Friendly**: Semantic HTML and meta tags
- **Accessibility**: Keyboard navigation and semantic structure

## 📋 Sections

1. **Header/Navigation**: Fixed navigation with responsive mobile menu
2. **Hero Section**: Eye-catching headline with CTA
3. **Mission Section**: Core mission and execution focus with isometric laptop illustration
4. **Why We Do It**: Vision statement and four key values
5. **Team Section**: Team members showcase
6. **Trust Section**: Three core pillars of excellence
7. **CTA Section**: Final call-to-action
8. **Footer**: Links, social media, and legal information

## 🛠️ Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **Vanilla JavaScript**: No dependencies
- **Font Awesome Icons**: Icon library via CDN

## 📦 Installation & Setup

### Option 1: Clone and Run Locally

```bash
# Clone the repository
git clone https://github.com/duckgamer23/creative-vanam-website.git

# Navigate to the project
cd creative-vanam-website

# Open in your browser (no build process needed!)
# Simply open index.html in your browser or use a local server
```

### Option 2: Using Python Local Server

```bash
# Python 3
python -m http.server 8000

# Then visit: http://localhost:8000
```

### Option 3: Using Node.js http-server

```bash
# Install http-server globally
npm install -g http-server

# Run the server
http-server

# Visit the provided URL
```

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #22c55e;        /* Green accent */
    --dark-bg: #0f172a;              /* Main background */
    --dark-secondary: #1e293b;       /* Secondary background */
    --dark-tertiary: #334155;        /* Tertiary background */
    --text-primary: #e2e8f0;         /* Primary text */
    --text-secondary: #cbd5e1;       /* Secondary text */
    --text-muted: #94a3b8;           /* Muted text */
}
```

### Content
Edit the text content directly in `index.html`:

- **Hero Section**: Lines 45-49
- **Mission Section**: Lines 65-75
- **Why We Do It**: Lines 110-155
- **Team Section**: Lines 160-180
- **Trust Section**: Lines 185-210
- **CTA Section**: Lines 215-225
- **Footer**: Lines 230-280

### Fonts
Modify the font-family in `styles.css`:

```css
body {
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', sans-serif;
}
```
